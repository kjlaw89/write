Write
=====

elementaryOS Writing application
Native application built on Granite, GTK, Vala and WebKit.

Dependencies:
gtk+-3.0 
granite 
webkitgtk-3.0 
javascriptcoregtk-3.0

To Build & Run:
Execute: ./quickbuild.sh
